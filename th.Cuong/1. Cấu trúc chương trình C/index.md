# **Cấu trúc chương trình C**

## **Câu 1:**

```c
#include <stdio.h>

int main(){
  printf("Hello, world!");
  return 0;
}
```

```c
    printf("     *\n");

    printf("   *****\n");
    printf("***********\n");
    printf("    * *\n");
    printf("    * *\n");
    return 0;
}
```
